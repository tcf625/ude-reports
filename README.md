# UDE - Reports 說明文件與範例

所有相關範例，目前需於公司內網執行，以取得相關 libraries

Maven Repository Setting : 

```xml
<repositories>
    <repository>
        <id>IISI_UDE</id>
        <url>http://192.168.57.21/nexus/content/groups/public</url>
        <releases>
            <enabled>true</enabled>
        </releases>
        <snapshots>
            <enabled>true</enabled>
            <updatePolicy>interval:5</updatePolicy>
        </snapshots>
    </repository>
</repositories>
```
